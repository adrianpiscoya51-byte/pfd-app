# app-ads.txt
# Archivo de ejemplo para declarar los vendedores autorizados (app-ads.txt).
# Reemplace las líneas de ejemplo por las entradas reales que le proporcionen AdMob / Ad Manager.
# Formato por línea: <domain>, <seller_id>, <type>, <cert_authority_id>
# - <domain>: dominio del sistema publicitario (ej. google.com)
# - <seller_id>: su ID de editor (ej. pub-1234567890123456)
# - <type>: DIRECT o RESELLER
# - <cert_authority_id>: identificador de la autoridad de certificación (opcional; para Google es f08c47fec0942fa0)
# Ejemplos:
google.com, pub-9045393905919240, DIRECT, f08c47fec0942fa0
#openx.com, 123456789, RESELLER, 9b7f9f6a...

#    (el archivo debe responder con HTTP 200 y contenido en texto plano)
